*** NO SE AÑADEN LOS SPRITES DE EMOJIS PARA REDUCIR ESPACIO ***
Este directorio contiene un conjunto reducido de emojis para la versión WASM,
seleccionados a partir del directorio "emojis".

Para el binario publicado se han incluido únicamente aquellos emojis cuyo tamaño
es menor o igual a 1 KByte, con el objetivo de reducir el tamaño final del 
archivo ".wasm".



*** EMOJI SPRITES NOT INCLUDED TO REDUCE REPOSITORY SIZE ***
This directory contains a reduced set of emojis for the WASM version,
selected from the main "emojis" directory.

Only emojis with a file size of 1 KByte or less were included in the published
binary, in order to minimize the final ".wasm" file size.
